# 펜타호-DB 연결하기

펜타호에 여러 종류의 데이터베이스(DB)들을 연결할 수 있다.

1. 해당 DB로부터 데이터를 가져와서 활용할 수 있다.
2. 데이터 처리를 마친 후에 이 데이터들을 원하는 DB에 저장할 수 있다.

펜타호에 연결할 수 있는 DB의 종류들은 매우 많다. 대표적인 DB들로는 아래와 같다.

- Cloudera Impala
- Google BigQuery
- Hadoop Hive
- IBM DB2
- MS SQL Server
- Maria DB
- MySQL
- Oracle
- PostgreSQL
- SQLite
- ...
